# Tweeter Project

Tweeter is a simple, single-page Twitter clone. It allows users to create tweets. It tracks who posted the tweet, and when it was posted.

## Dependencies

- body-parser 1.15.2
- Express
- Node 5.10.x or above
- change 1.02
- db: 1.0.6
- md5: 2.1.0
- moment: 2.18.1
- mongo: 0.1.0
